# ML-exam

## Task

This project has the aim to classify and cluster Stars, Quasars and Galaxies with the highest possible accuracy with the models and according to the metrics learned in the Machine Learning course of University of Verona, taken in A.Y. 2025/26.

## Repository Content

This repository contains all the material needed to run and test the code provided inside `src`.
A technical report is available inside the `documentation` folder with all the results, elaboration and conclusions drawn from the development of this project.

## Tested Algorithms
- Supervised models: KNN, Random Forest and Support Vector Machines
- Unsupervised models: K-Means and Gaussian Mixture Models (GMM)

Read all about them in the documentation!
